RealtyPHP-WP is a free, open source real estate listing plugin for Wordpress.

== Description ==

RealtyPHP-WP is a free real estate listing plugin for WordPress. Available languages: English(US, GB) and Hungarian. Easy to install and
maintain. No special skills are required to manage listings. The administrator and the registered user can manage listings, add and
delete photos for listing. The administrator can manage the countries, and the property types.
Author: Elod Horvath(http://www.tunedthemes.com)

== Installation ==

Requirements:
Wordpress 3.3 or above, 
PHP Version 5.2.17 or above, 
Mysql database with innoDB support

1. Upload the downloaded, and unzipped `realtyphp` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
